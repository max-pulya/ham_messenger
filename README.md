# HAM MESSENGER SMS BY MAXPULYA

Приложение, выжимающее максимум из смс. Поддерживает шифрование шифром вернама с абсолютной криптостойкостью. Внутри приложения ипользуется кодировка cp1251, что позволяет передавать 140 русских символов как 1 смс. Планируется добавить сжатие. 

Для транспорта закодированное смс преобразовывается в валидную строку кодировки gsm7bit.

# СХЕМА ПРЕОБРАЗОВАНИЙ
Ввод->байты cp1251->шифрование xor ключом(опционально)->строка gsm7bit->отправка текстового смс. У получателя программа делает обратное преобразование.

У получателя сообщение отображается в стандартном приложении смс, однако он тоже должен установить данное приложение 


Приложение имеет два ярлыка, один для отправки смс, другой для дешифровки зашифрованных
Прием смс из данного приложения не тестировался на ios и работа приложения была проверена только на устройствах андроид.

# Начало работы, выдача разрешений
Скачать и установить апк можно в releases.
Для начала работы с приложением необходимо открыть через ярлык "отправить", и в ходе отправки первого смс выдать разрешение на управление файлами и доступ к sms.
Андроид 15 может не давать активировать разрешение, необходимо зажать по ярлыку -> о приложении -> Три точки -> Разрешить доступ к настройкам

# Активация шифрования
1)Создайте папку ham_keys в storage/emulated/0 (внутреннее хранилище) на обоих устройствах.

2)Сгенерируйте два файла с случайными данными, один для отправки от абонента A к B, а другой для отправки от B к A.

3)Скопируйте первый файл под именем send+7номер_B (например если номер абонента b +78005553535, то файл должен называться send+78005553535. Обязательно пишите номер с +7) на устройство абонента A и на устройство абонента B под именем receive+7номер_A в папку ham_keys.

4)Скопируйте второй файл под именем send+7номер_B на устройство абонента A и на устройство абонента B под именем receive+7номер_A в папку ham_keys.
